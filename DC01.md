# DC01 — Domain Controller

Windows Server 2022, the lab's Active Directory domain controller. Lives on the MGMT VLAN.

## Config

| Property      | Value                                |
|---------------|--------------------------------------|
| Hostname      | DC01                                  |
| OS            | Windows Server 2022                   |
| VLAN          | 10 (MGMT)                             |
| IP            | 10.10.10.10/24                        |
| Gateway       | 10.10.10.1 (OPNsense MGMT)            |
| DNS           | 10.10.10.10 (self)                    |
| Domain        | `corp.internal`                       |
| FSMO roles    | All five (single-DC lab)              |

## Active Directory

5 Organizational Units under `DC=corp,DC=internal`:

- IT Infrastructure
- Security Operations
- Development
- Human Resources
- Finance

11 domain users provisioned via PowerShell over WinRM (see `Build-Log.md` 2026-04-13 entry, "AD OUs and Users"). All accounts use a uniform lab password and are enabled.

Domain-joined clients:

- Windows 11 (10.10.10.100, DHCP)

## Remote Management

WinRM enabled on `http://10.10.10.10:5985/wsman`, NTLM auth. PowerShell can be run from the Linux host using `pywinrm` — that's how the OUs and users were provisioned without ever opening the virt-manager console.
